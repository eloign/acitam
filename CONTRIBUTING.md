# Contributing to these notes

Thank you for your interest in collaborating on these notes!

This repository gathers small notes on the various ways mathematics can be applied from a pure mathematician's perspective. Meaning, the style is chosen to be rigorous and formal, yet it should not prevent us from exploiting humour throughout the notes. The only thing we chose to omit from rigorousness is graphomania. So, we expect many a proof to be given in the form of an algorithm highlighting its main revelations. 

These notes are intended to grow into a book called "A Comprehensive Introduction to Applicable Mathematics", if God permits.

# How you can help

1. **Report errors or suggestions** by opening an issue. Among other, please, point out typos, factual mistakes, missing references, or unclear passages.
2. **Propose additions** by submitting a pull request with new sections, examples, or applications that fit the project’s scope.
